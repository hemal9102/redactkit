# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good intent, focus on the work. Harassment,
personal attacks, and discriminatory comments are not tolerated.

## Reporting

Email **support@corenovus.com** with a short description of the incident.
Reports are kept confidential. We respond within 5 business days.

For security issues (vulnerabilities, not behaviour) use the process in
[SECURITY.md](SECURITY.md) instead.
